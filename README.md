# 🛡️ Phishing Awareness Training

An interactive web-based training module that educates users about phishing attacks, social engineering tactics, and online safety best practices. Features real-world examples and an interactive quiz.

---

## ✨ Features

- 🎨 **Modern Dark UI** — Premium glassmorphism design with smooth animations
- 📚 **Educational Content** — Covers what phishing is, how it works, and who's targeted
- 🔍 **6 Attack Types** — Email phishing, spear phishing, whaling, smishing, vishing, clone phishing
- 📧 **Real-World Examples** — Interactive fake email and fake website with annotated red flags
- 🛡️ **8 Protection Tips** — Best practices to stay safe online
- 🧠 **Interactive Quiz** — 10-question quiz with instant feedback and scoring
- 📱 **Fully Responsive** — Works on desktop, tablet, and mobile

---

## 🚀 How to Run

Simply open `index.html` in any modern web browser:

```bash
# Option 1: Open directly
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows

# Option 2: Use a local server (recommended)
python3 -m http.server 8000
# Then visit http://localhost:8000
```

No dependencies, no build step, no frameworks — just pure HTML, CSS, and JavaScript.

- Incident response steps
